# ToDo Core Wpf

## Description

Simple ToDo (task management) tool.

## Requirement

- Windows 11 Pro 20H1+
- .NET 5
- Visual Studio 2019

## Usage

Build this with Visual Studio 2019 and run.

## Install

Fork and clone this repository.

```
$ git clone git@github.com:yourname/ToDoCoreWpf.git
```
